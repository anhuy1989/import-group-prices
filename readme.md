Description
-----------
Click System/Import Export/ Import Custom Tool
Import group prices from CSV file:
CSV format:  s

| sku     | website_id | group_id | price |
|---------|------------|-------------------|-------|
| 1231231 | 1          | 3                 | 120   |

------------
- PHP >= 5.2.0
- Mage_Core
- ...

Installation Instructions
-------------------------

    modman deploy Anhi_ImportGroupPrices
